## Early-up info.

To view the commands, you need the `Manage Message` Permission.

## Start an giveaway

use `/giveaway start` to start a giveaway.

## End a giveaway

use `/giveaway end` to end a giveaway.

## reroll a giveaway

use `/giveaway reroll` to reroll a giveaway.
